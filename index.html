1. Scope of Work

1.1 Project Description

The Developer agrees to design, develop, and deliver a Minimum Viable Product (MVP) Customer Relationship Management (CRM) system for the Client as detailed in the Statement of Work dated 14 July 2025.

View the full Statement of Work: https://kaoriumsow.netlify.app/

1.2 Key Deliverables

The MVP shall include:

  • Multi-tenant authentication system for retail partners
  • Customer data segmentation by date, fragrance preferences, and descriptive words
  • Email interface with scheduling capabilities
  • GDPR-compliant data storage in EU region (Supabase or equivalent)
  • Data import/export functionality
  • Basic analytics dashboard

1.3 Technical Specifications

Frontend: Next.js 14/15, TypeScript, Tailwind CSS
Backend: Supabase (EU region), PostgreSQL
Email Services: Resend API or equivalent
Hosting: Vercel or equivalent
